# Testing and Staged Rollout

## Internal testing
1. Create an Internal testing track and upload the AAB
2. Add testers by email group; share opt-in link
3. Submit for review if required; wait for processing
4. Verify install, login, lessons flow, rewards, and notifications

## Pre-launch report
- Enable automated device testing in Play Console
- Review crashes, ANRs, permission warnings, deep links, performance

## Closed testing (optional)
- Promote the release to Closed testing for a broader audience
- Update release notes and iterate
