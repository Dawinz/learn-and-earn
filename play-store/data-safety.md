# Data Safety (Guide)

This guide helps you complete the Google Play Data safety form.

## Data collected
- Personal info: Email (for account/support)
- App activity: Lesson progress, streaks
- Device or other IDs: Advertising ID (if ads)

## Data shared
- With service providers (analytics/crash) and ad partners (if applicable)

## Data handling
- Data is encrypted in transit
- Data can be deleted upon request
- Data is processed to provide core features and analytics

Note: Adapt based on actual SDKs used (e.g., Firebase Analytics, Crashlytics, Google Mobile Ads). Keep this file updated with any new SDKs or purposes.
