# Play Store Submission Kit

This folder contains everything needed (besides the AAB) to publish the Learn & Earn Android app on Google Play.

Contents:
- `checklist.md` — end‑to‑end steps from prep to production rollout
- `store-listing.md` — app name, short/long descriptions, categorization, and release notes template
- `privacy-policy.md` — hosted policy text you can publish as a page
- `data-safety.md` — guidance for the Data safety form
- `content-rating.md` — guidance for the content rating questionnaire
- `assets.md` — required image assets, sizes, and screenshot guide
- `signing.md` — Android app signing/keystore guidance
- `flutter-release.md` — Flutter build steps to generate a release AAB
- `test-rollout.md` — internal/closed testing, pre-launch report and staged rollout
- `release-notes.md` — templates/examples for “What’s new” per track
- `contacts.md` — support email, website, and contact details

Recommended order:
1) Fill `contacts.md` and `store-listing.md`
2) Review `privacy-policy.md` and host it (link in listing)
3) Complete `data-safety.md` and `content-rating.md` on the Play Console
4) Prepare assets from `assets.md`
5) Follow `signing.md` and `flutter-release.md` to build and upload
6) Use `test-rollout.md` for testing and rollout
